# Digital Nurture - Module 1

## Project Name
Factory Design Pattern Implementation

## Description
This project demonstrates the Factory Design Pattern in Java.
It includes creation of different document types using a common factory.

## Technologies Used
- Java

## Modules

### Exercise 1
- Logger implementation using Singleton-like behavior

### Exercise 2
- Document interface
- WordDocument, PdfDocument, ExcelDocument
- DocumentFactory class
- Main class for execution

## How to Run
1. Compile all Java files
2. Run Main.java

## Output Example
Opening Word Document  
Opening PDF Document  
Opening Excel Document
